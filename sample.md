# Welcome to MDEV! 🎉

This is a **native desktop** markdown editor built with PyQt5.

## Features

- ✅ Live preview
- ✅ Syntax highlighting
- ✅ Full markdown support
- ✅ File management

### Text Formatting

You can write **bold text**, *italic text*, or ~~strikethrough~~.

Here's some `inline code` and a code block:

```python
def hello():
    print("Hello, World!")
```
